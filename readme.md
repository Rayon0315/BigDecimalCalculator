- # BigDecimalCalculator

  BigDecimalCalculator 是一个基于 **C++ / Qt** 实现的高精度大整数计算器项目。
   项目从底层自定义大整数数据结构出发，实现表达式解析、运算规则处理与图形界面交互，适合作为工程化 C++ 学习与算法实践项目。

  ------

  ## 项目简介

  在标准 C++ 中，`int` 和 `long long` 存在位数限制。本项目通过自定义大整数结构，实现任意长度整数运算，并构建完整的表达式解析与求值流程。

  项目特点：

  - 支持任意长度整数
  - 支持带括号的中缀表达式
  - 表达式合法性检测
  - 中缀转后缀（逆波兰表达式）求值
  - Qt 图形界面交互

  ------

  ## 核心功能

  - 大整数加法
  - 大整数减法
  - 大整数乘法
  - 大整数除法
  - 取模运算
  - 表达式语法校验
  - 运算符优先级处理
  - 括号嵌套计算

  ------

  ## 项目结构说明

  项目整体分为三个层级：

  1. 数值层
      实现大整数数据结构及基础运算逻辑。
  2. 表达式层
      负责表达式合法性校验、中缀转后缀以及后缀求值。
  3. 界面层
      基于 Qt 构建图形界面，与计算引擎解耦。

  这种分层方式有利于后续扩展，例如增加新运算符或更换前端界面。

  ------

  ## 构建方式

  ### 环境要求

  - Qt 5 或 Qt 6
  - CMake 3.10+
  - 支持 C++11 及以上标准的编译器

  ### 构建流程

  1. 克隆仓库
  2. 使用 Qt Creator 打开 CMakeLists.txt
  3. 选择对应 Qt Kit
  4. 构建并运行

  或使用命令行构建：

  - 创建 build 目录
  - 运行 cmake 配置
  - 编译生成可执行文件

  ------

  ## 技术实现思路

  ### 大整数实现

  - 使用自定义数据结构存储数字
  - 按位进行运算
  - 处理符号位
  - 支持任意位数扩展

  ### 表达式处理流程

  用户输入 → 合法性校验 → 中缀转后缀 → 后缀表达式求值 → 输出结果

  表达式解析与计算逻辑分离，使结构清晰、可维护。
